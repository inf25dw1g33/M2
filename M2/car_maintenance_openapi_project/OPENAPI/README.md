# Trabalho Prático - Car Maintenance API

Estrutura base do projeto para a UC **Informática Desenvolvimento Web I**.

- `openapi.yaml` — especificação OpenAPI 3.0 da API.
- `express-server/` — implementação Node.js/Express da API.
- `db/` — definição da base de dados MySQL (Dockerfile + `init.sql`).
- `docker-compose.dev.yml` — ambiente de desenvolvimento (build local).
- `docker-compose.prod.yml` — ambiente de produção (usa imagens do Docker Hub).
